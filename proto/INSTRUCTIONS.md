## Instructions for preparing protos
### Prerequisites 
- Add  GOPATH/bin `$(go env GOPATH)/bin` to your os PATH
- Have Python 3 with Pip package manager installed
- Golang 1.18++

## Prepare
Linux/Unix/Mac
```shell
chmod +x ./prepare.sh
./prepare.sh
```
Windows/DOS
```shell
./prepare.bash
```

## Compile
```shell
make
```
